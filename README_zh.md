# 结印 Ketsuin - 忍者手势输入法

[English Documentation](README.md)

**Ketsuin (结印)** 拒绝凡人的键盘敲击，带你重返**忍者的世界**。以此项目，将古老的**结印**之术与现代**计算机视觉**（YOLOX-Nano）完美融合，打造出这一款 T9 结印输入系统。

凝聚查克拉，结出十二地支之印，像真正的忍者一样将意念转化为文字。该项目基于 **ONNX Runtime Web** 驱动的神经网络，实时捕捉你的每一个手势，将摄像头化身为通灵之术的媒介。

![Ketsuin Demo](public/asset/demo.gif)

## 功能特性

-   **手势检测**: 使用轻量级 YOLOX-Nano 模型通过 ONNX Runtime 实时检测 12 种不同的手势。
-   **T9 忍者引擎**: 实现严格的 T9 输入系统，每个手势对应传统电话键盘上的一个按键。
-   **移动端优化**: 完全响应式布局，完美支持桌面和移动设备。
-   **连续删除**: 长按“酉”印可快速删除文本。
-   **防误触**: 智能输入延迟（普通按键 100ms，空格键 40ms），有效防止手势抖动导致的误触。

## 使用说明

### 十二地支 (The 12 Signs)

该输入法使用 12 个手势（子、丑、寅等）映射到按键 1-9, *, 0, #。

| 手势 ID | 名称 (印) | T9 按键 | 功能 / 字母 |
| :--- | :--- | :--- | :--- |
| **1** | **子 (Rat)** | **1** | `. , ? ! : ; 1` |
| **2** | **丑 (Ox)** | **2** | `A B C 2` |
| **3** | **寅 (Tiger)** | **3** | `D E F 3` |
| **4** | **卯 (Hare)** | **4** | `G H I 4` |
| **5** | **辰 (Dragon)** | **5** | `J K L 5` |
| **6** | **巳 (Snake)** | **6** | `M N O 6` |
| **7** | **午 (Horse)** | **7** | `P Q R S 7` |
| **8** | **未 (Ram)** | **8** | `T U V 8` |
| **9** | **申 (Monkey)** | **9** | `W X Y Z 9` |
| **10** | **酉 (Bird)** | **(*)** | **删除 / 退格** (长按连续删除) |
| **11** | **戌 (Dog)** | **(0)** | **空格 / 确认候选词** |
| **12** | **亥 (Boar)** | **(#)** | **下一个候选词 / 切换** |

### 基本操作

1.  **结印**: 向摄像头展示手势。
2.  **输入**: 对应的数字将添加到您的序列中。
3.  **候选**: 匹配序列的单词将会出现。
4.  **切换**: 使用 **亥 (#)** 印切换候选词。
5.  **确认**: 使用 **戌 (0)** 印确认选择的单词并插入空格。
6.  **删除**: 使用 **酉 (*)** 印删除最后一个字符。

## 开发

### 前提条件

-   Node.js
-   npm

### 本地运行

```bash
npm install
npm run dev
```

### 技术栈

-   **React + TypeScript + Vite**: 前端框架。
-   **ONNX Runtime Web**: 在浏览器中运行 YOLOX 目标检测模型。
-   **Tailwind CSS**: 样式和响应式设计。
