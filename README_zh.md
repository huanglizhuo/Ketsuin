# 结印 Ketsuin - 忍者手势输入法

[English Documentation](README.md)

**Ketsuin (结印)** 是一种实验性的输入法，它使用**忍者手势**（基于火影忍者中的结印）代替传统键盘。它利用计算机视觉实时检测手势，并将其映射到 T9 预测文本系统。

## 功能特性

-   **手势检测**: 使用轻量级 YOLOX-Nano 模型通过 ONNX Runtime 实时检测 12 种不同的手势。
-   **T9 忍者引擎**: 实现严格的 T9 输入系统，每个手势对应传统电话键盘上的一个按键。
-   **预测文本**: 基于输入序列和内置字典提供单词建议。
-   **反向映射**: 手动输入文本时，动态显示对应的结印序列。
-   **移动端优化**: 完全响应式布局，完美支持桌面和移动设备。
-   **连续删除**: 长按“酉（鸟）”印可快速删除文本。

## 使用说明

### 十二地支 (The 12 Signs)

该输入法使用 12 个手势（子、丑、寅等）映射到按键 1-9, *, 0, #。

| 手势 ID | 名称 (印) | T9 按键 | 功能 / 字母 |
| :--- | :--- | :--- | :--- |
| **1** | **子 (Rat)** | **1** | `. , ? ! : ; 1` |
| **2** | **丑 (Ox)** | **2** | `A B C 2` |
| **3** | **寅 (Tiger)** | **3** | `D E F 3` |
| **4** | **卯 (Hare)** | **4** | `G H I 4` |
| **5** | **辰 (Dragon)** | **5** | `J K L 5` |
| **6** | **巳 (Snake)** | **6** | `M N O 6` |
| **7** | **午 (Horse)** | **7** | `P Q R S 7` |
| **8** | **未 (Ram)** | **8** | `T U V 8` |
| **9** | **申 (Monkey)** | **9** | `W X Y Z 9` |
| **10** | **酉 (Bird)** | **(*)** | **删除 / 退格** (长按连续删除) |
| **11** | **戌 (Dog)** | **(0)** | **空格 / 确认候选词** |
| **12** | **亥 (Boar)** | **(#)** | **下一个候选词 / 切换** |

### 基本操作

1.  **结印**: 向摄像头展示手势。
2.  **输入**: 对应的数字将添加到您的序列中。
3.  **候选**: 匹配序列的单词将会出现。
4.  **切换**: 使用 **亥 (#)** 印切换候选词。
5.  **确认**: 使用 **戌 (0)** 印确认选择的单词并插入空格。
6.  **删除**: 使用 **酉 (*)** 印删除最后一个字符。

## 开发

### 前提条件

-   Node.js
-   npm

### 安装与运行

```bash
npm install
npm run dev
```

### 技术栈

-   **React + TypeScript + Vite**: 前端框架。
-   **ONNX Runtime Web**: 在浏览器中运行 YOLOX 目标检测模型。
-   **Tailwind CSS**: 样式和响应式设计。
